# Bank Churn Analysis
Data analysis project exploring customer churn using Python (Pandas) in Google Colab. Analysed customer behaviour to identify key factors influencing churn, including activity levels and account balance. Conducted data quality checks, analysis, and data interpretation.
## Overview
This project analyses a bank customer dataset to identify patterns linked to customer churn.

## Tools Used
- Python (Pandas)
- Google Colab
- Kaggle dataset

## Key Insights
- Inactive customers were significantly more likely to churn
- Customers with higher balances showed higher churn rates
- Customer engagement appears to be a key factor in retention

## Data Considerations
The dataset contained no missing values, but may not include factors such as customer satisfaction or service quality. These limitations were considered when interpreting results.

## Outcome
This project demonstrates how data analysis can be used to generate insights and support data-driven decision making in a financial context.
